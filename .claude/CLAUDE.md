## Skills

커스텀 검증 및 유지보수 스킬은 `.claude/skills/`에 정의되어 있습니다.

| Skill | Purpose |
|-------|---------|
| `verify-implementation` | 프로젝트의 모든 verify 스킬을 순차 실행하여 통합 검증 보고서를 생성합니다 |
| `manage-skills` | 세션 변경사항을 분석하고, 검증 스킬을 생성/업데이트하며, CLAUDE.md를 관리합니다 |
| `verify-react-patterns` | React 컴포넌트 구조, hooks 사용 패턴, 명명 규칙을 검증합니다 |
| `verify-supabase` | Supabase 사용 패턴, 에러 처리, 쿼리 최적화를 검증합니다 |
| `verify-auth` | 인증 및 보안 패턴, ProtectedRoute 사용, 권한 체크를 검증합니다 |
| `verify-imports` | import 순서, 경로 일관성, 사용하지 않는 import를 검증합니다 |
